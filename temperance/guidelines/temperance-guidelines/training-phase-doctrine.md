# Training Phase Doctrine

Status: invariant core companion.

## Purpose

This file defines what each generic phase is trying to do and how periodization should change emphasis across a build.

Use it to answer:
- what the phase is for
- what should usually be emphasized
- what should not be lost completely
- how intensity mix should behave across the phase

The event overlay still defines what `specific` means.
The philosophy overlay still biases the exact stimulus mix.

## Core periodization idea

Periodization should shift **emphasis**, not create total amnesia.

That means:
- each phase has a main job
- each phase should still preserve occasional contact with non-dominant qualities when that helps the next phase
- the planned weekly load budget still governs how much of each element can appear

Important rule:
- a phase should not collapse the whole block into one lane
- some general mix of intensities and structural elements should still appear every now and then

## 1) Return / Re-entry

### Main job

- Re-establish tolerable contact with the primary specific modality.
- Rebuild consistency, frequency, and clean short-horizon recovery.

### Usual emphasis

- easy and recovery-oriented work
- structural rhythm
- conservative progression of the primary specific load

### What should not be lost completely

- enough total load to avoid needless detraining when the athlete can support it
- light touches of higher-end work only when they improve coordination, confidence, or structure without distorting the phase

### Main risk

- trying to force build-phase stress before basic tolerance is back

## 2) Base / Capacity Build

### Main job

- Expand the athlete's usable range.
- Raise the ceiling while deepening aerobic support and structural durability.

### Default periodization pattern

This phase should usually explore the **extremes**, not live in one monotonous middle lane.

In practical terms, that often means:
- a lot of easy / low-intensity work to build deep aerobic adaptation, durability, and recoverable volume
- some intentional ceiling-touch work to keep top-end development alive and reveal room above current sustainable output
- enough structural discipline that the whole mix still fits inside the planned weekly load budget

Important clarification:
- exploring the extremes does **not** mean random hero sessions
- it means widening the range of usable capacities while keeping the week absorbable

### Usual emphasis

- large aerobic support
- durability-building volume
- productive but controlled higher-end touches
- honest spacing between meaningful stressors

### What should not be lost completely

- some threshold / event-relevant work now and then, so the build does not become disconnected from later specificity
- some moderate and transitional work where it helps structure rather than filling space meaninglessly

### Main risks

- everything becomes medium
- ceiling work disappears entirely
- easy volume becomes junk
- durability work is sacrificed while chasing only top-end expression

## 3) Specificity

### Main job

- Converge the build toward the demands that most determine event performance.

### Usual emphasis

- more event-specific hard work
- progression of the event's key duration or specific anchor
- tighter alignment between weekly structure and event demands

### What should not be lost completely

- enough easy support to keep the structure recoverable
- occasional touches of non-dominant qualities so the athlete does not become flat or fragile

### Main risk

- narrowing too early or too aggressively, so specificity becomes expensive rather than productive

## 4) Peak

### Main job

- Protect and express the best work already built.

### Usual emphasis

- stabilizing the highest useful specific work
- sharpening
- reducing unnecessary randomness

### What should not be lost completely

- enough support load and rhythm to keep the athlete feeling like a trained athlete rather than a detraining athlete

### Main risk

- continuing to build when consolidation would produce more usable performance

## 5) Taper

### Main job

- Reduce fatigue while preserving readiness.

### Usual emphasis

- lower total load
- maintained rhythm
- selective intensity touch
- lower mechanical or total fatigue cost

### What should not be lost completely

- confidence
- event feel
- enough intensity contact that the athlete does not feel dulled

### Main risk

- confusing freshness with inactivity

## Cross-phase reminder

- The dominant element should change by phase.
- The full set of useful elements should not disappear for too long without a clear reason.
- Weekly anchors remain the budget that decides how much can actually fit.
