# Training History Memo

Status: repo-safe template.

## Purpose

This file is the tracked, repo-safe template for long-term athlete history and evidence.

If `training-history-memo.local.md` exists, use that file instead for the real athlete history.

Use the history memo as:
- evidence for athlete-state interpretation
- long-range context for overlay revisions
- background when recent active-build state alone is not enough

Do **not** use the history memo as the invariant doctrine source of truth.

## Suggested structure

### Big picture

- athlete background
- historical load capacity
- current asymmetry between engine and durability, if relevant

### Long-term data summary

- notable recurring load patterns
- meaningful peak weeks or months
- major outlier experiments

### Build history

For each important build:
- what worked
- what failed
- likely lessons
- what should or should not be reused

### Injury and interruption history

- key injuries
- likely contributing patterns
- what altered training successfully preserved fitness

### Cross-build synthesis

- what seems to work best
- what repeatedly creates problems
- what should influence the current athlete-state overlay

### Current overlay implications

- what this history implies for the current athlete-state interpretation
- what should tighten progression or alert sensitivity
- what should remain background evidence only

## Update note

Keep real athlete-specific history in `training-history-memo.local.md` when it should remain private or unpushed.
